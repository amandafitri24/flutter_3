# 🌸 Flower Bouquet App  
### Modern Flutter-Based Flower Ordering Application

<p align="center">
  A beautifully designed mobile application built with Flutter for showcasing and ordering flower bouquets with an elegant and modern user interface.
</p>

---

## 📌 Overview

Flower Bouquet App is a mobile application developed using Flutter that simulates a simple e-commerce experience for flower bouquet ordering. The application focuses on delivering a visually appealing interface combined with smooth navigation and interactive user experience.

This project was developed as part of the Mid-Term Examination (UTS) for Mobile Application Development.

---

## 🎯 Objectives

The main objectives of this project are:

- To implement a mobile application using Flutter
- To design a modern and aesthetic user interface
- To implement multi-screen navigation
- To ensure all user interactions (buttons) function correctly
- To simulate a basic ordering system without backend/database integration

---

## ✨ Key Features

### 🏠 Product Catalog
Users can browse a list of flower bouquet products displayed with images, names, and prices in a clean and elegant layout.

### 📄 Product Details
Each product includes a detailed view showing:
- High-quality product image  
- Product name  
- Price information  
- Short description  

### 🛒 Order System
Users can interactively:
- Increase or decrease quantity  
- Instantly see calculated total price  

### 💳 Checkout Simulation
Displays customer information including:
- Full name  
- Phone number  
- Address  

---

## 🔄 Application Flow

The application follows a simple and intuitive flow:

1. User opens the application  
2. Browses flower products  
3. Selects a product  
4. Views product details  
5. Proceeds to order  
6. Adjusts quantity  
7. Views total price  
8. Proceeds to checkout  

---

## 🎨 User Interface & Design Approach

This application emphasizes modern UI/UX principles, including:

- Full-screen background image  
- Dark overlay for better contrast  
- Glassmorphism effect (blur UI)  
- Smooth animations (fade & scale)  
- Responsive layout for mobile devices  

The design aims to provide a premium and aesthetic user experience similar to modern e-commerce applications.
